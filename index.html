CallMagic Tweak

Oriented towards calls either over Cellular or Internet, this tweak is designed to give the user control over the flowing features:

  • Play music during call (implemented)
  • Record call (in progress)
  • Change voice during call (in progress)
  • Answering machine (in progress)
  • Incoming call view redesign (in progress)
  • More to be added based on your requests

This tweak was tested on devices with iOS 10 up to 11.3.1 and it is still in development.

In order to activate the tweak please support the development effort and make a donation of minimum 3$ using the Paypal link below.

Version 1.0:

  • Added call detection.
  • Added music player UI.
  • Added audio mixing functionality for headphone, speaker and mic.
  • Added preferences table for default settings.
